#### Function to fill missing values

``` r
fill_missing = function(vector) {
  if (is.numeric(vector)) {
    v_mean = mean(vector, na.rm=TRUE)
    vector[is.na(vector)] = v_mean
  }
  if (is.character(vector)) {
    vector[is.na(vector)] = "virginica"
  }
  return(vector)
}
```
